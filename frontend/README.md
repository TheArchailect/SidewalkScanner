# Frontend (React + Vite + TS)

A modern, and lean development framework for web projects consisting of a dev server via Vite allowing quick Hot Module Replacement, code bundling with Rollup, and component-based UI with React.

## Prerequisites

```bash
# All Node/NPM installs happen inside SideWalkScanner\frontend
# Install React + Vite + TS
cd frontend
npm install

# Launch Vite dev server HMR
npm run dev
```

## Scripts

- `npm run dev`     - Start Vite dev server with "Hot Module Replacement"
- `npm run build`   - Build production bundle
- `npm run preview` - Preview production build locally
- `npm run lint`    - Run ESLint checks
- `npm run test`    - Run frontend tests
